Multisensor

  • Multisensor Linked Data: web presentation 2014-05-19, Barcelona
  • NIF-example.ttl: example of NLP data in RDF (Turtle). Covers NIF (text binding), OLIA (linguistic properties), Penn (POS tagging), Stanford (dependency parsing), ITS20 (semantic annotation properties), NERD (entity extraction classes), Stanbol/FISE (multiple NLP tools/annotations per word/phrase), MARL (opinion/sentiment); and use of entities from DBpedia, WordNet, YAGO.
  • NIF-example.json: same in JSON-LD, shows that Turtle should be used for examples/discussion/QA and JSON-LD for machine communication only

SPARQL Syntax Diagrams

Syntax (rail-road) diagrams of SPARQL 1.1 Query Language. I use this quite often while writing SPARQL

  • sparql11-grammar.xhtml: Cross-linked diagrams, one per production (173 total). A bit hard to understand: use this for reference, but not for learning SPARQL. Here's a chunked diagram, which however is partial: Discussion whether there's a chunked generator
  • sparql11-grammar.ebnf: EBNF rules extracted from the spec
  • sparql11-grammar.txt: same thing but with production numbers
